# ADT Blog source

This branch holds the Hugo source for the ADT blog, published at
<https://awslabs.github.io/awsome-distributed/>.

This branch is editorially curated by AWS authors. External PRs targeting
this branch are auto-closed; please open an issue with feedback or topic
suggestions instead.

The companion code lives on
[`main`](https://github.com/awslabs/awsome-distributed).

## Local preview

```bash
git submodule update --init --recursive
hugo server -D
```

Requires Hugo extended ≥ 0.140.2.
